Flask API

This project demonstrates the creation of an API using Python,Flask and MySQL.
MySQL connects to Python, and the database tables are exposed as Restful API that supports CRUD operations(REST,POST,PUT,DELETE).
Postman was used to validate the API, along with the framework Pytest to test the endpoints automatically.
